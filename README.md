#Internet Usage Analysis

A Python data analysis project that explores internet usage trends across selected African countries and the global average using World Bank data.

#Project Objectives

- Clean and preprocess real-world data using pandas.
- Analyze internet usage trends over time.
- Create informative visualizations with Matplotlib.
- Compare internet adoption across countries.
- Demonstrate a complete data analysis workflow.

#Technologies

- Python
- pandas
- Matplotlib
- NumPy

#Dataset

The dataset was obtained from the World Bank Open Data platform and contains internet usage indicators for countries and regions across multiple years. "data.worldbank.org" (https://reference-url-citation.invalid/0)
